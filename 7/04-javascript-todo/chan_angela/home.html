Welcome to your To Do list

Features

  1. Mousing over an item in a list will make it change color to help you focus on where your mouse is.
  2. You can add an activity by using the textbox below and then clicking Submit Activity. This will add your activity to your to do list.
  3. Clicking on an activity in the to do list will bring it over to the done list.
  4. Clicking on an activity in the done list will delete the activity.
  5. WARNING: If you make a mistake, you must bring it over to the done list and delete it from there!

Happy planning!
